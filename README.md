# Ejercicios de Python

Este repositorio contiene una colección de ejercicios prácticos diseñados para aprender y reforzar conceptos de Python. Los ejercicios van desde básicos hasta intermedios, incluyendo:

- Entrada y salida de datos (`input` y `print`)  
- Condicionales (`if`, `elif`, `else`)  
- Bucles (`for`, `while`)  
- Listas y diccionarios  
- Validación de datos  
- Cálculos y estadísticas simples  

Cada ejercicio está comentado y listo para ejecutar, lo que permite practicar paso a paso y entender cómo funcionan las estructuras de Python.
